時空傳送指南 (Grammar Rule)

搞懂這三大條件句，修復時空超輕鬆！

🕒

Zero Conditional (Type 0) - 事實與常理

用於科學真理、習慣或不變的事實（恆常如此）。

If / When + Present Simple, + Present Simple
// If you heat ice, it melts.
🔮

First Conditional (Type 1) - 未來可能性

用於在未來**極可能**發生的事情（有很大機率實現）。

If + Present Simple, + will + Infinitive (動詞原形)
// If it rains tomorrow, we will stay at home.
💭

Second Conditional (Type 2) - 虛擬幻想

與**現在事實相反**的白日夢、幻想、或機率極低的設想。

If + Past Simple, + would/could + Infinitive
// If I won the lottery, I would buy a spaceship.
// 注意: Type 2中，If子句的Be動詞一律傾向用 "were" (e.g. If I were you...)
